# 🌴 Sri Lanka Travel App

A beautiful, interactive Flutter application designed to help users discover and explore popular travel destinations across Sri Lanka. The app provides curated information about stunning locations, along with an intuitive interface and the ability to save favorite destinations.

## ✨ Features

- **Home Screen:** Discover popular destinations and recently explored places categorized elegantly.
- **Detailed Information:** View comprehensive details about each location, including descriptions and ratings.
- **Save Favorites:** Easily bookmark your favorite spots and view them anytime on the dedicated Favorites screen.
- **Profile Layout:** A clean user profile interface.
- **Smooth Navigation:** Seamlessly switch between different app sections using a custom bottom navigation bar.

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

- Flutter SDK (latest version): [Install Flutter](https://docs.flutter.dev/get-started/install)
- A code editor like VS Code or Android Studio.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/travel_app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd travel_app
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
